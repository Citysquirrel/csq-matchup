# 🔍 포켓몬 상성 돋보기 (Pokemon Type Matchup)

![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react&logoColor=61DAFB)

![Vite](https://img.shields.io/badge/Vite-b39aff?style=flat&logo=vite&logoColor=white)

**포켓몬 상성 돋보기**는 포켓몬스터 초보 유저들을 위한 **웹 기반 직관적 상성 분석 대시보드**입니다. 적군과 아군 포켓몬을 배치하여 복잡한 타격/피격 상성(x4, x2, x0.5, x0 등)을 한눈에 파악하고, 최적의 교체 타이밍과 기술을 결정할 수 있도록 돕습니다.

## ✨ 핵심 기능 (Key Features)

- 🛡️ 실시간 상성(타격/피격) 배지 시스템

- 🔍 간단한 검색 시스템

- 🧹 정제된 포켓몬 데이터베이스

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend:** React, TypeScript, CSS3
- **Data Handling:** Node.js (Data processing scripts)
- **Assets:** PokeAPI, Pokemon Showdown Server (Sprites & Audio)
- **Build Tool:** Vite
